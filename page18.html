'Autoémpleate ofreciendo tus servicios
de diseñador web para móviles'.

Mobirise Website Builder es una aplicación freeware de diseño web desarrollada por Mobirise que permite que los usuarios creen y publiquen sitios web sin saber programar ni diseño gráfico.

Ha sido presentado como una alternativa offline para los populares sistemas de creación de sitio web como Wix.com, Weebly, Jimdo, Webydo y Squarespace.

La primera versión fue lanzada en 2015 y estaba enfocada a crear sitios web sin saber programación y en cumplir con los estándares de diseño web adaptable de Google. EN 2015, se lanzó la versión 2.0, que agregó menús desplegables, formularios de contacto, animaciones y soportes para temas y extensiones de terceros. Desde la versión 3.0 se han añadido nuevos temas.

PROGRAMA

> De este curso:

MODULO 1 : El Diseño Web

Se tratan algunos aspectos básicos de las diferentes maneras de diseñar un sitio web, cómo ha evolucionado en el tiempo y las tendencias actuales.

MODULO 2 : El software Mobirise

20 min                        

En este módulo se analizan las características del software o programa MOBIRISE destacando los beneficios y ventajas para crear sitios web orientados a dispositivos móviles.

MODULO 3 : Cómo instalar y configurar Mobirise

1 hora

Se detallan los pasos a seguir para descargarse, instalar y configurar adecuadamente el programa en un ordenador tanto para WINDOWS como MAC.


MODULO 4 : Cómo crear un sitio web con MOBIRISE (I)

1 hora

En un primer acercamiento, se presenta la metodología de creación de sitios web usando el programa MOBIRISE. Desde la elección del nombre del sitio hasta su publicación en internet para que sea accesible al mundo.


MODULO 5 : Cómo crear un sitio web con MOBIRISE (II).

1 hora
40 min

Se trata el tema de las plantillas de diseño que incluye MOBIRISE y cómo seleccionarlas para cada sitio web. Cómo crear las diferentes páginas que tendrá el sitio web. También se trata el uso de los bloques que incluyen cada plantilla, cómo insertarlos, como modificarlos y cómo moverlos de lugar.



MODULO 6 : Cómo crear un sitio web con MOBIRISE (III).

1 hora

En este módulo se trata de enlazar adecuadamente a todas las páginas del sitio para permitir la correcta navegación por el sitio. También se trata el tema de las secciones de COOKIES, AVISO LEGAL y PRIVACIDAD que obligatoriamente hay que incluir en un sitio web.



MODULO 7 : Cómo publicar el sitio Web con MOBIRISE (I).

1 hora

Aqui tratamos las diferentes alternativas para publicar fácilmente un sitio web usando MOBIRISE. 



MODULO 8 : Cómo publicar el sitio Web con MOBIRISE (II).

1 hora

Se detalla una alternativa que permite disponer de alojamiento de calidad gratuito, con subdominio propio y certificado de seguridad SLL. También se analiza como configurar un dominio propio.


MODULO 9 : Cómo planificar la creación de un sitio Web.

1 hora

En este módulo se plantea y detalla la estrategia típica de creación de un sitio web, desde la idea  original, el planteamiento, el desarrollo, el diseño, la publicación y la promoción.



MODULO 10 : Cómo gestionar en el dia a dia un sitio Web.

1 hora

Finalmente tratamos cómo tratar con los aspectos típicos que ocurren durante la gestión diaria de un sitio web que necesita ser modificado, agrandado o eliminado.
